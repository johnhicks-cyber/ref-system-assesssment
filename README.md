# RMF System Assessment Project

## Overview
This project demonstrates implementation of the NIST Risk Management Framework (RMF) for a sample federal information system.

## Framework Used
- NIST SP 800-37 (RMF)
- NIST SP 800-53 (Security Controls)
- FISMA guidelines

## Steps Performed
1. Categorized system based on impact level
2. Selected applicable security controls
3. Assessed control implementation
4. Identified gaps and risks
5. Recommended remediation actions

## Key Outcome
Developed a System Security Plan (SSP) and control assessment aligned with federal compliance requirements.

## Skills Demonstrated
- Risk Assessment
- NIST RMF
- Vulnerability Analysis
- Security Monitoring
